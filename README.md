# skyeye

#### 项目介绍
win10风格的一套系统，刚开始开发，内容慢慢添加，已完成用户的删除，查看，编辑功能。</br>
##### 启动方式
在项目skyeye上右键选用maven-build，在Goals中输入命令tomcat7:run，启动完成后，访问http://localhost:8081即可。
#### 目前功能

- 菜单管理
- 用户管理
- 角色管理
